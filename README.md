# 项目实训文档页

这个仓库用于发布公开可访问的项目实训文档页。

访问地址：

<https://fengqizihang.me>

## 日常更新

主要修改 `index.md`。

修改完成后提交并推送到 `main` 分支，GitHub Pages 会自动发布到上面的域名，通常几十秒到几分钟后生效。

## 文件说明

- `index.md`：网页正文，日常主要编辑这个文件。
- `_layouts/default.html`：网页样式和基础布局，一般不用改。
- `_config.yml`：GitHub Pages/Jekyll 配置，一般不用改。
- `CNAME`：自定义域名绑定文件，不要删除。
